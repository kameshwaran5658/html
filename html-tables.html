HTML Tables

HTML tables are used to present tabular data. They are defined using the <table> element and consist of rows and columns.

Basic Structure

A table is created with the <table> tag and contains rows (<tr>) and cells. Each row can contain header cells (<th>) and data cells (<td>).

Example

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Attributes

  • border: Specifies the width of the table border.
  • cellpadding: Defines the space between cell borders and cell content.
  • cellspacing: Sets the space between cells.

Usage

Tables are useful for displaying structured data, such as schedules, financial reports, and comparison charts.

Conclusion

Understanding HTML tables is essential for presenting data in a clear and organized manner on web pages.
