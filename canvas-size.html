How to Set the Height and Width of the Canvas in HTML5

The <canvas> element in HTML5 allows you to draw graphics using JavaScript. You can set the height and width of the canvas using HTML attributes or CSS.

Using HTML Attributes

<canvas width="500" height="300"></canvas>

Using CSS

<style>
    .canvas-style {
        width: 500px;
        height: 300px;
    }
</style>

<canvas class="canvas-style"></canvas>

Conclusion

Setting the dimensions of the canvas element ensures that your graphics are displayed at the desired size. You can choose between HTML attributes for a fixed size or CSS for more flexibility.
