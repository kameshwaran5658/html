HTML <ul> Tag

The <ul> tag defines an unordered list. An unordered list can have bullet points as list markers.

Example

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Attributes

  • type: Specifies the type of bullet to use (disc, circle, square).

Usage

The <ul> tag is used when the order of the items is not important, such as a list of features or random notes.

Conclusion

Unordered lists are useful for displaying items without implying a sequence, making them ideal for lists where the order does not matter.
