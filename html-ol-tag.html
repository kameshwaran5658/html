HTML <ol> Tag

The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.

Example

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Attributes

  • type: Specifies the kind of marker to use (numbers, letters, etc.).
  • start: Defines the starting value of the list items.
  • reversed: Reverses the order of the list items.

Usage

The <ol> tag is used when the order of the items is important, such as steps in a process or ranking.

Conclusion

Ordered lists are useful for displaying items in a specific sequence, providing a clear and structured way to present information.
