How to Set the Start Value of an Ordered List in HTML5

The <ol> tag in HTML5 creates an ordered list. You can set the starting value of the list using the start attribute.

Example

<ol start="5">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Conclusion

The start attribute allows you to specify where the numbering begins in an ordered list.
