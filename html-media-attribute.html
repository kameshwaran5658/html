HTML Media Attribute

The media attribute is used with the <link> and <style> elements to specify the media type that the linked resource or style is designed for. It helps in applying styles or resources conditionally based on the media type.

Example

<link rel="stylesheet" href="print.css" media="print">
<link rel="stylesheet" href="screen.css" media="screen">

Usage

  • media="print": Applies the stylesheet when the document is printed.
  • media="screen": Applies the stylesheet when the document is viewed on a screen.

Conclusion

The media attribute is essential for responsive design and ensuring that styles are applied appropriately based on the device or medium being used.
