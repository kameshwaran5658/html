HTML Audio Tag

The <audio> element is used to embed audio content in a web page. It allows for native audio playback and supports multiple audio formats.

Basic Usage

The <audio> tag supports various attributes to control audio playback. You can specify multiple <source> elements to provide different audio formats for better browser compatibility.

Example

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>

Attributes

  • controls: Displays playback controls such as play, pause, and volume.
  • autoplay: Starts playback automatically when the page loads.
  • loop: Repeats the audio when it finishes.
  • muted: Mutes the audio.
  • preload: Specifies how the browser should load the audio. Values include auto, metadata, and none.

Conclusion

The <audio> element provides a simple and effective way to embed audio content in web pages, with built-in controls and support for various audio formats.
