HTML Src Attribute

The src attribute specifies the URL of the media file. It is commonly used with <img>, <audio>, and <video> tags.

Example

<img src="image.jpg" alt="Description of image">

Usage

The src attribute is essential for embedding media files, providing the source location of the content.

Conclusion

The src attribute is fundamental for embedding images, audio, and video content into HTML documents.
