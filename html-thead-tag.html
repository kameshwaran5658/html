HTML <thead> Tag

The <thead> tag groups the header content in an HTML table. It is used in conjunction with <tbody> and <tfoot> to define different sections of a table.

Example

<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
</table>

Usage

The <thead> tag is useful for defining the header rows of a table, which helps with styling and scripting.

Conclusion

Using the <thead> tag improves table organization and readability by clearly separating header content.
