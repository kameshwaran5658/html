HTML <form> Tag

The <form> tag is used to create a form for user input. It can contain various input elements and is used to collect data from users.

Attributes

  • action: Specifies the URL where the form data will be sent when the form is submitted.
  • method: Defines the HTTP method (GET or POST) used to send form data.
  • enctype: Specifies how the form data should be encoded (e.g., application/x-www-form-urlencoded, multipart/form-data).
  • target: Specifies where to open the response (e.g., _self, _blank, _parent).

Example

<form action="/submit" method="post">
    <input type="text" name="username" placeholder="Enter your username">
    <input type="submit" value="Submit">
</form>

Usage

The <form> tag is crucial for creating interactive web pages that require user input and data submission.

Conclusion

The <form> tag is the foundation for user interaction on websites, enabling data collection and submission.
