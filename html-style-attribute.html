HTML Style Attribute

The style attribute allows you to add CSS styles directly to an element. This is known as inline styling.

Example

<p style="color: blue;">This is a blue paragraph.</p>

Usage

While the style attribute is useful for quick styling, it's generally better to use external or internal CSS for more maintainable code.

Conclusion

The style attribute is a powerful tool for inline styling, but should be used sparingly for maintainability.
