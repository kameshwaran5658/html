HTML Canvas Complete Reference

The <canvas> element is used to draw graphics on the fly via JavaScript. It provides a space where you can create and manipulate images, shapes, and animations directly within your web page.

Basic Structure

The <canvas> element itself does not display any graphics; it provides a blank space where graphics can be drawn using JavaScript.

Example

<canvas id="myCanvas" width="200" height="200"></canvas>
<script>
    var canvas = document.getElementById('myCanvas');
    var ctx = canvas.getContext('2d');

    // Draw a red rectangle
    ctx.fillStyle = 'red';
    ctx.fillRect(10, 10, 150, 100);

    // Draw a blue circle
    ctx.beginPath();
    ctx.arc(100, 150, 50, 0, Math.PI * 2);
    ctx.fillStyle = 'blue';
    ctx.fill();
    </script>

Common Canvas Methods

  • getContext(type): Returns a drawing context on the canvas.
  • fillRect(x, y, width, height): Draws a filled rectangle.
  • beginPath(): Starts a new path.
  • arc(x, y, radius, startAngle, endAngle): Draws an arc.
  • fill(): Fills the current path.

Usage

The <canvas> element is used for creating games, graphs, and other visual content that requires dynamic and interactive graphics.

Conclusion

Mastering the <canvas> element allows for the creation of rich, interactive graphics and animations directly within web applications.
