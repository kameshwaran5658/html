HTML Block and Inline Elements

HTML elements are typically either block-level elements or inline elements.

Block-level Elements

Block-level elements typically start on a new line and take up the full width available. Examples include:

  • <div>
  • <h1> to <h6>
  • <p>
  • <form>

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples include:

  • <span>
  • <a>
  • <img>
  • <strong>

Example

<div>This is a block-level element.</div>
<span>This is an inline element.</span>

Conclusion

Understanding the difference between block and inline elements is essential for structuring and styling your HTML documents.
