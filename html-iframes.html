HTML Iframes

An HTML iframe is used to display a web page within a web page. The <iframe> tag specifies an inline frame.

Example

<iframe src="https://www.example.com" width="600" height="400"></iframe>

Attributes

Common attributes used with iframes include:

  • src - Specifies the URL of the page to embed.
  • width - Specifies the width of the iframe.
  • height - Specifies the height of the iframe.
  • frameborder - Specifies whether or not to display a border around the iframe.

Conclusion

Iframes are useful for embedding other web pages or content such as videos within a web page.
