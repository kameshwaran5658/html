HTML <input> Tag

The <input> tag is used to create interactive controls in web forms. It supports various types of input, such as text, passwords, checkboxes, and more.

Types of Input

  • text: Single-line text input.
  • password: Single-line text input, hides the text.
  • checkbox: Checkable box, allowing multiple selections.
  • radio: Radio buttons, allowing a single selection from a group.
  • submit: Submit button for form submission.
  • file: File selection control.
  • date: Date picker control.
  • number: Numeric input control.

Example

<input type="text" name="username" placeholder="Enter your username">

Attributes

  • type: Specifies the input type (e.g., text, password, checkbox).
  • name: Defines the name of the input field, used for form submission.
  • value: Specifies the default value of the input field.
  • placeholder: Provides a hint about the expected value of the input.
  • required: Specifies that the input field must be filled out before submitting the form.

Usage

The <input> tag is versatile and used for various types of data entry in web forms.

Conclusion

The <input> tag is fundamental for creating forms and collecting user input in HTML.
