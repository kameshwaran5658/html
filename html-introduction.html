HTML Introduction

Welcome to the HTML Introduction section. Here, you will learn the basics of HTML, the standard markup language for creating web pages.

What is HTML?

HTML stands for HyperText Markup Language. It is used to create web pages and web applications. HTML describes the structure of a web page using markup. HTML elements are the building blocks of HTML pages.

HTML Elements

HTML elements are represented by tags. Each element consists of a start tag, content, and an end tag. For example:

<p>This is a paragraph.</p>

Basic HTML Document Structure

An HTML document has a defined structure. Here is an example of a basic HTML document:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

HTML Tags

HTML tags are the core building blocks of an HTML document. Tags usually come in pairs, an opening tag and a closing tag. The content is placed between these tags.

Example of HTML Tags

  • <h1></h1> - defines a top-level heading
  • <p></p> - defines a paragraph
  • <a></a> - defines a hyperlink
  • <img> - defines an image (self-closing tag)

Conclusion

Now you have a basic understanding of what HTML is and how it is used to create web pages. In the next section, we will dive deeper into HTML basics and explore more HTML tags and elements.
