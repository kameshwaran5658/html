HTML <td> Tag

The <td> tag defines a cell in an HTML table. It is used inside a row (<tr>) to hold data or content for that cell.

Example

<table>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Attributes

  • colspan: Specifies the number of columns a cell should span.
  • rowspan: Defines the number of rows a cell should span.

Usage

The <td> tag is essential for defining individual cells in a table and arranging data accordingly.

Conclusion

Proper use of the <td> tag ensures that table data is displayed clearly and effectively.
