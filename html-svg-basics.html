HTML SVG Basics

Scalable Vector Graphics (SVG) is an XML-based format for vector graphics. SVG allows you to create graphics and animations that are resolution-independent and can be scaled without loss of quality.

Basic Structure

SVG elements are defined using the <svg> tag. Within this tag, you can include various SVG shapes, paths, and text elements.

Example

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
    <rect x="10" y="10" width="30" height="30" stroke="black" stroke-width="3" fill="blue" />
</svg>

Common SVG Elements

  • <circle>: Draws a circle.
  • <rect>: Draws a rectangle.
  • <line>: Draws a straight line.
  • <path>: Defines a complex shape or path.
  • <text>: Adds text to the SVG graphic.

Usage

SVG is ideal for creating logos, icons, and complex illustrations that need to scale across different devices and screen resolutions.

Conclusion

Understanding the basics of SVG enables you to create high-quality, scalable graphics that are perfect for modern web applications.
