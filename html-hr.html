HTML hr Tag

The <hr> tag is used to define a thematic break in an HTML page (e.g., a shift of topic).

Example

<hr>

Usage

Typically used to separate content or define a change in content.

Conclusion

The <hr> tag is a simple way to add a horizontal line to your web page to separate content.
