How to Create an Unordered List in HTML

An unordered list in HTML is created using the <ul> tag. Each item in the list is defined with the <li> tag.

Example

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Conclusion

The <ul> tag creates a bulleted list, which is useful for listing items where the order is not important.
