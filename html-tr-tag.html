HTML <tr> Tag

The <tr> tag defines a row in an HTML table. It is used to group data cells (<td>) and header cells (<th>) into a row.

Example

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Usage

The <tr> tag helps in organizing data into rows, making it easier to read and manage table content.

Conclusion

Using the <tr> tag effectively arranges data in rows, enhancing the readability of tables.
