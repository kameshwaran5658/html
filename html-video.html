HTML Video

The <video> element is used to embed video content in a web page. It provides built-in controls for play, pause, and volume, and supports various video formats.

Basic Usage

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    Your browser does not support the video tag.
</video>

Attributes

  • controls: Adds playback controls.
  • autoplay: Starts playing the video automatically.
  • loop: Loops the video.
  • muted: Mutes the video.
  • width, height: Specifies the dimensions of the video player.

Conclusion

The <video> element is a powerful tool for embedding video content on web pages, providing a range of options for playback and user interaction.
