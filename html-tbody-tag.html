HTML <tbody> Tag

The <tbody> tag groups the body content in an HTML table. It is used to define the main data rows of the table, separating them from the header and footer sections.

Example

<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Footer 1</td>
            <td>Footer 2</td>
        </tr>
    </tfoot>
</table>

Usage

The <tbody> tag helps with styling and scripting by grouping all the body rows of the table separately from the header and footer.

Conclusion

The <tbody> tag improves the structure and manageability of table data.
