HTML Layout

In this section, we will explore the layout structure of HTML documents. Understanding layout is essential for designing web pages and organizing content effectively.

HTML Document Structure

The basic structure of an HTML document includes the following elements:

  • <html></html> - Root element that contains the entire HTML document.
  • <head></head> - Contains meta-information about the document, such as title and link to stylesheets.
  • <body></body> - Contains the content of the document that is displayed in the web browser.

HTML Layout Elements

HTML provides several elements to structure your web page layout:

  • <header></header> - Defines the header section of the document, often used for navigation, branding, etc.
  • <nav></nav> - Defines a navigation section with links.
  • <main></main> - Defines the main content area of the document.
  • <section></section> - Defines a section within the document, used to group related content.
  • <article></article> - Defines a self-contained article or piece of content.
  • <aside></aside> - Defines content that is tangentially related to the content around it.
  • <footer></footer> - Defines the footer section of the document, often used for copyright and contact information.

Example Layout

Here is an example of a basic HTML layout using these elements:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>HTML Layout Example</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>My Website</h1>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section id="home">
            <h2>Welcome to My Website</h2>
            <p>This is the home section.</p>
        </section>

        <section id="about">
            <h2>About Me</h2>
            <p>This is the about section.</p>
        </section>

        <section id="contact">
            <h2>Contact</h2>
            <p>This is the contact section.</p>
        </section>
    </main>

    <footer>
        <p>© 2024 My Website. All rights reserved.</p>
    </footer>
</body>
</html>

Responsive Layout

To make your HTML layout responsive and work on different screen sizes, you can use CSS media queries. Here's an example:

@media (max-width: 768px) {
    header nav ul {
        flex-direction: column;
    }
}

Conclusion

Understanding HTML layout is crucial for creating well-structured and organized web pages. With these layout elements, you can effectively manage and present your content.
